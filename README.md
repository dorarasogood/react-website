# 店家商品資訊網站
提供店家展示活動訊息、商品價格的一個資訊網站。<br/>
展示網站: [https://dorarasogood.github.io/react-website/#/](https://dorarasogood.github.io/react-website/#/)
<br/>
<br/>

特色:

1. 多國語系(i18n)
2. 深色模式(Dark Mode)
3. 響應式(RWD)
4. 單元測試(Unit Test)

<br/>

|        |       |
|  ----  | ----  |
| library| React |
| 語言   | JavaScript |
| CSS框架| Bootstrap |
| 後端   | Express |
